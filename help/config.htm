Editor de configuración

El editor permite gestionar los servidores y sus listas de puertos. Se abre desde la barra de herramientas principal pulsando Editor de configuración.

Barra de herramientas del editor

La barra se divide en tres grupos separados por divisores:

BotónDescripción
Nuevo servidor Limpia el formulario para introducir un servidor nuevo. El servidor no se crea hasta pulsar Guardar.
Guardar Crea el servidor nuevo o actualiza el servidor seleccionado. Se activa sólo cuando hay cambios pendientes.
⎯ Separador
Importar CSV Importa servidores desde un archivo CSV. Los servidores se añaden a los existentes.
Exportar CSV Exporta todos los servidores a un archivo CSV.
⎯ Separador
Agregar puerto Abre el diálogo Puerto para añadir un nuevo puerto al servidor seleccionado.
Editar puerto Abre el diálogo Puerto con los datos del puerto seleccionado. Requiere una fila seleccionada.
Borrar puerto Elimina el puerto seleccionado de la lista. Requiere una fila seleccionada.
Consejo: Pase el ratón sobre cualquier botón para ver su descripción en el tooltip.

Formulario de servidor

Situado inmediatamente debajo de la barra de herramientas:

  • Nombre — texto identificativo del servidor (obligatorio).
  • IP — dirección IPv4 con máscara xxx.xxx.xxx.xxx (obligatorio).
  • Tipo — perfil de puertos predeterminados (ver tabla).

Nuevo servidor: Pulse Nuevo servidor, rellene los campos y pulse Guardar. Se creará una pestaña nueva con los puertos predeterminados del tipo elegido.

Pestañas de servidor

Cada servidor tiene su propia pestaña con la cuadrícula de puertos. Haga clic derecho sobre una pestaña para eliminar ese servidor (el propio menú contextual actúa como confirmación).

Lista de puertos

Muestra los puertos del servidor activo: Puerto, Protocolo y Descripción. Haga clic en la cabecera Puerto o Protocolo para ordenar ▲/▼.

Menú contextual (clic derecho sobre la lista)

  • Clic sobre una fila existente → Editar puerto  /  Borrar puerto
  • Clic en área vacía → Agregar puerto

Diálogo de puerto

Al pulsar Agregar o Editar se abre el diálogo Puerto:

  • Puerto — número entero entre 1 y 65 535 (solo dígitos).
  • Protocolo — TCP o UDP.
  • Descripción — texto libre. Al salir del campo Puerto con la descripción vacía, se rellena automáticamente si el puerto es conocido (DNS 53, LDAP 389, RDP 3389…).

Pulse Aceptar para confirmar o Cancelar para descartar.

Importar / Exportar CSV

Formato: hostname,ip,type,puerto/proto,…

hostname,ip,type,ports (port/proto)
DC01,10.0.0.1,DC,88/TCP,88/UDP,389/TCP,445/TCP
SRV-PRINT,10.0.0.5,PrintServer,80/TCP,443/TCP,9100/TCP

Tipos válidos: DC · PrintServer · SCCM · SCCM_DP

Importar: Los servidores del CSV se añaden a los ya existentes. Las líneas con menos de 4 columnas o puertos fuera de rango se omiten y se contabilizan en el resumen final.

Timeout y confirmación

El selector Timeout (esquina inferior izquierda) fija el tiempo máximo de espera por conexión: 500 ms, 1 000 ms o 2 000 ms.

Pulse Aceptar para aplicar todos los cambios, o Cancelar para descartarlos.
