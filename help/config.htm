Editor de configuración

El editor permite gestionar los servidores y sus listas de puertos. Se abre desde la barra de herramientas principal (Editor de configuración) o pulsando Ctrl+E.

Barra de herramientas

La barra se divide en tres grupos separados por divisores:

BotónDescripción
□ Nuevo servidor Limpia el formulario para introducir un servidor nuevo. El servidor no se crea hasta pulsar Guardar.
💾 Guardar Crea el servidor nuevo o actualiza el servidor seleccionado con los datos del formulario. Se activa solo cuando hay cambios.
—Separador
⇓ Importar CSV Importa servidores desde un archivo CSV. Los servidores importados se añaden a los existentes.
⇑ Exportar CSV Exporta todos los servidores a un archivo CSV.
—Separador
+ Agregar puerto Abre el diálogo Puerto para añadir un nuevo puerto al servidor seleccionado.
✎ Editar puerto Abre el diálogo Puerto con los datos del puerto seleccionado para modificarlo. Requiere seleccionar una fila.
🗑 Borrar puerto Elimina el puerto seleccionado. Requiere seleccionar una fila.
Consejo: Pase el ratón sobre cualquier botón de la barra para ver su descripción en el tooltip.

Formulario de servidor

Situado inmediatamente debajo de la barra de herramientas:

  • Nombre — texto identificativo del servidor (obligatorio).
  • IP — dirección IPv4 con máscara xxx.xxx.xxx.xxx (obligatorio).
  • Tipo — perfil de puertos predeterminados: DC, PrintServer, SCCM, SCCM DP.
Nuevo servidor: Pulse Nuevo servidor, rellene el formulario y pulse Guardar. Se creará una pestaña nueva con los puertos predeterminados del tipo elegido.

Pestañas de servidor

Cada servidor ocupa una pestaña. Haga clic derecho sobre una pestaña para eliminar ese servidor (se pide confirmación implícita en el propio menú).

Lista de puertos

La lista muestra los puertos del servidor seleccionado con tres columnas: Puerto, Protocolo y Descripción. Haga clic en la cabecera de Puerto o Protocolo para ordenar ▲/▼.

Menú contextual (clic derecho)

  • Clic derecho sobre una fila → Editar puerto / Borrar puerto.
  • Clic derecho en área vacía → Agregar puerto.

Diálogo de puerto

Al agregar o editar un puerto se abre el diálogo Puerto con los campos:

  • Puerto — número entero entre 1 y 65 535 (solo dígitos).
  • Protocolo — TCP o UDP.
  • Descripción — texto libre. Al salir del campo Puerto con la descripción vacía, se rellena automáticamente si el puerto es conocido (DNS 53, LDAP 389, RDP 3389, etc.).

Pulse Aceptar para confirmar o Cancelar para descartar.

Importar / Exportar CSV

Formato de columnas: hostname,ip,type,puerto/proto,…

hostname,ip,type,ports (port/proto)
DC01,10.0.0.1,DC,88/TCP,88/UDP,389/TCP,445/TCP
SRV-PRINT,10.0.0.5,PrintServer,80/TCP,443/TCP,9100/TCP

Tipos válidos: DC · PrintServer · SCCM · SCCM_DP

Importar: Los servidores importados se añaden a los ya existentes; no se eliminan los actuales. Las líneas con menos de 4 columnas o puertos fuera de rango se omiten.

Timeout y aceptar cambios

El selector Timeout (esquina inferior izquierda) establece el tiempo máximo de espera por conexión: 500 ms, 1 000 ms o 2 000 ms.

Pulse Aceptar para aplicar todos los cambios a la configuración activa, o Cancelar para descartarlos.
