Editor de configuración

Barra de acciones

La parte superior del editor contiene:

  • Nuevo servidor — limpia el formulario para introducir un servidor nuevo.
  • Nombre — texto identificativo del servidor.
  • IP — dirección IP destino (control con máscara xxx.xxx.xxx.xxx).
  • Tipo — tipo de servidor (DC, PrintServer, SCCM, SCCM DP).
  • Guardar / Actualizar — desactivado hasta detectar cambios. Si hay pestaña seleccionada, actualiza ese servidor; si no, crea uno nuevo.

Pestañas de servidor

Cada servidor tiene su propia pestaña con la cuadrícula de puertos. Clic derecho sobre una pestaña muestra la opción para eliminar ese servidor.

Cuadrícula de puertos

  • Agregar puerto — introduzca número, protocolo y descripción en la parte inferior.
  • Actualizar — modifique una fila seleccionada.
  • Eliminar puerto — borra la fila seleccionada.
  • Clic en cabecera Puerto o Protocolo para ordenar ▲/▼.
Descripción automática: Al salir del campo Puerto, la descripción se rellena automáticamente si el puerto es conocido (DNS, LDAP, SMB, RDP, etc.).
