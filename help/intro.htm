Introducción

NetChecker es una herramienta de diagnóstico de red para administradores de sistemas Windows. Verifica la conectividad TCP/UDP hacia puertos específicos de servidores de infraestructura empresarial.

Soporta los siguientes tipos de servidor con sus perfiles de puertos por defecto:

  • Domain Controller (DC) – Kerberos, LDAP, SMB, DNS, RPC…
  • Print Server – HTTP, SMB, LPD, IPP, RAW…
  • SCCM Full – HTTP, HTTPS, SMB, WSUS, Remote Control…
  • SCCM Distribution Point – HTTP, HTTPS, SMB, WSUS
Requisito: Windows x64. No requiere instalación.
