Primeros pasos

  1. Inicie NetChecker.exe. Si no existe configuración, se abre el Editor de configuración automáticamente.
  2. Añada los servidores que desea comprobar.
  3. Revise los puertos de cada servidor y active los que necesite.
  4. Pulse Aceptar para guardar.
  5. Pulse Comprobar para iniciar el test.
Consejo: La IP de origen se detecta automáticamente y no se guarda en el archivo de configuración.
